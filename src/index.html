Uniões - Sites de Casamento

Next.js React.js TypeScript Tailwind CSS

Plataforma "white-label" para casais criarem e gerenciarem sites de casamento completos, com painel administrativo, personalização e lista de presentes.

O Desafio

Criar uma plataforma "white-label" que permitisse a casais, mesmo sem conhecimento técnico, construir e gerenciar um site de casamento completo, centralizando informações, personalização e interações com convidados em um único lugar.

A Solução

Desenvolvi a "Uniões", uma aplicação web com um painel administrativo poderoso e intuitivo. A solução permite personalização total do site (fotos, textos, cores) e gerencia funcionalidades essenciais como lista de presentes (físicos e PIX), checklist de tarefas e controle de orçamento, tudo salvo localmente no navegador do usuário para uma experiência rápida e sem necessidade de banco de dados.

Culinária Inteligente com IA

Node.js IA API React.js TypeScript

Ferramenta de gestão para chefs que automatiza cálculos nutricionais, gera listas de compras e organiza fichas técnicas usando IA.

O Desafio

Projetar uma aplicação web robusta para chefs e entusiastas da gastronomia, com o objetivo de otimizar a gestão de criações culinárias, automatizando tarefas complexas como cálculos nutricionais e planejamento de compras através da inteligência artificial.

A Solução

O "Culinária Inteligente" é uma ferramenta de gestão completa que serve como um centro de comando para a organização de receitas (fichas técnicas), ingredientes e planejamento. A plataforma se destaca por funcionalidades como cálculo nutricional automático via IA, importação inteligente de dados, geração de listas de compras otimizadas e uma comunidade integrada para compartilhamento de criações.

Principais Funcionalidades

  • Gestão de Fichas Técnicas: Criação e armazenamento de receitas detalhadas.
  • Cálculo Nutricional Automático: Análise nutricional completa por IA.
  • Importação Inteligente: Extração de dados de PDF e código para cadastro rápido.
  • Lista de Compras Inteligente: Geração de listas otimizadas com cálculo de custos.
  • Comunidade Integrada: Espaço para compartilhar e descobrir receitas.
  • Exportação de Dados: Fichas e listas exportáveis para PDF, TXT e CSV.

Conversor TXT para Excel

Python Tkinter Pandas SQLite

Aplicação desktop com Python/Tkinter que converte arquivos TXT para Excel, incluindo um sistema de login e cadastro com banco de dados SQLite.

Conversor .Txt para XLSX

O Desafio

Desenvolver uma aplicação de desktop segura e eficiente para converter arquivos de texto (TXT) em planilhas do Excel. O sistema deveria incluir autenticação de usuários, com funcionalidades de cadastro e login, para garantir o acesso controlado à ferramenta.

A Solução

Criei um conversor de arquivos utilizando Python com a biblioteca Tkinter para a interface gráfica. A aplicação conta com um sistema de gerenciamento de usuários (cadastro e login) que utiliza um banco de dados SQLite para armazenar as credenciais de forma segura. Após a autenticação, o usuário pode selecionar um arquivo TXT, que é processado com a biblioteca Pandas e exportado para o formato XLSX (Excel).

Principais Funcionalidades

  • Autenticação Segura: Sistema completo de cadastro e login de usuários.
  • Interface Gráfica Intuitiva: Desenvolvida com Tkinter para uma experiência de desktop amigável.
  • Armazenamento Local: Credenciais salvas de forma segura em um banco de dados SQLite.
  • Conversão Eficiente: Utiliza a biblioteca Pandas para processar os dados do arquivo TXT.
  • Exportação para Excel: Gera um arquivo no formato XLSX com os dados processados.
